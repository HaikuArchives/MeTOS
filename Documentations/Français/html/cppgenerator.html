Creer et Ecrire le code C++...



Pour generer le code sous MeTOS, vous devez utiliser un genrateurs.
Ces derniers sont accessibles par le menu Generateurs.

Actuellement uniquement le Generateur C++ est fonctionnel.


Le generateur C++ affiche pares un clique sur le bouton Demarrer le resultat des fichiers generees.

Quand un objet a la propriete Derivee, un fichier source et en-tete sera genere.
Pour Ecrire ces fichiers sur le disque-dur, cliquez sur le bouton Ecrire.
Les fichiers seront ecrit dans le chemin specifie dans l'objet BProject.
Dans cette version le makefile est genere avec les fichiers, mais il est possible de desactive cette option dans les preferences de MeTOS.
