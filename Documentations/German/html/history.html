Warum erstelle ich MeTOS...



Ich startete dieses Projekt im Jahr 2001, weil alle vorhandenen Projekte eine Bibliotheken (Libraries) Anhängigkeit besaßen.
Das Be API ist sauber und kann ohne spezielle Bibliotheken genutzt werden.
Als ich die erste Version veröffentlichte, traf ich auf viele Probleme, desshalb schrieb ich den Quellcode im Mai 2001 neu.
Die derzeitige Version ist noch in der Entwicklung, aber es ist möglich eine GUI zu erstellen und C++ Code zu schreiben.

Sicher, das Projekt enthält einige BUGs (Fehler), aber ich arbeite alleine an diesem Projekt und es kostet viel Zeit meine Idee zu verwirklichen.
Aber mit Hilfe von den Benutzerinformationen (feedback), konnte ich schon viele Fehler und Problembereiche beheben oder verbessern,
so das ich mein Projekt erhebich verbessern konnte.
